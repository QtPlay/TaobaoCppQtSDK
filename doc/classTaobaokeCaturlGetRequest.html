Detailed Description

TOP API: 淘宝客类目推广URL.

由于受到淘宝联盟pid升级规则影响，该接口后续可能会下线。建议直接使用拼链接的方式（塞入完整的pid）来获取搜索结果，而不是调用该接口来获取结果

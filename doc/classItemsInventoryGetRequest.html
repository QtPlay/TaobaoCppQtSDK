Detailed Description

TOP API: 获取当前用户作为卖家的仓库中的商品列表，并能根据传入的搜索条件对仓库中的商品列表进行过滤 只能获得商品的部分信息，商品的详细信息请通过taobao.item.get获取

QString ItemsInventoryGetRequest::banner
private

分类字段。可选值:

regular_shelved(定时上架)
never_on_shelf(从未上架)
off_shelf(我下架的)
for_shelved(等待所有上架)

sold_out(全部卖完)
violation_off_shelf(违规下架的)
默认查询的是for_shelved(等待所有上架)这个状态的商品
<font color="red">注：for_shelved(等待所有上架)=regular_shelved(定时上架)+never_on_shelf(从未上架)+off_shelf(我下架的)</font>
